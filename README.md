# CRUD app with FastAPI, GraphQL and Masonite ORM

## How to use

1. Fork/Clone

1. Create virtual env and install requirements:

    ```sh
    $ poetry install
    ```

1. Update *config/database.py*

1. Run the server:

    ```sh
    $ poetry run uvicorn main:app
    ```

    Navigate to [http://localhost:8000](http://localhost:8000)
